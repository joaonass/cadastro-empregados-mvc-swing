# cadastro-empregados-mvc-swing
Projeto Java com Swing usando arquitetura MVC. Permite cadastro de empregados com nome, idade e matrícula, vinculando cada um a um departamento via JComboBox. Aplica orientação a objetos, encapsulamento, validação de dados e relacionamento entre classes.

## Funcionalidades

- Cadastro de empregados
- Seleção de departamento via JComboBox
- Associação de empregado a um departamento
- Validação de matrícula funcional
- Estrutura baseada em MVC

## Conceitos aplicados

- Programação Orientada a Objetos (POO)
- Encapsulamento
- Relacionamento entre classes (Empregado → Departamento)
- Arquitetura MVC
- Interface gráfica com Java Swing

## Estrutura do projeto

src/
├── model/        # Classes de domínio (Empregado, Departamento, Pessoa)
├── view/         # Interface gráfica (JanelaEmpregado)
├── controller/   # Controle de eventos e regras
└── Main.java     # Inicialização do sistema

## Como executar

1. Clone o repositório:

https://github.com/joaonass/cadastro-empregados-mvc-swing

2. Abra o projeto em uma IDE Java (NetBeans, IntelliJ ou Eclipse)
